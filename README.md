# guessTheNumber
# Proyecto de adivina el numero
Pequeño proyecto de adivinar el número.
En este sencillo juego se tiene que adivinar un número que lanza la página de 1 a 100.
Cada intento se marca y se cuenta con un reloj que va contando el tiempo.
Cuando se acierta al número se detiene el reloj y se muestra el número de intentos, hora y fecha en una tabla en la parte inferior.
Se usan métodos Math.random para enviar el número aleatoriamente.
Cuenta tambíen con links para redes sociales.
